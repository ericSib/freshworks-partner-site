## Notes techniques / decisions

### Prix reference

`SERVICE_PRICE_FROM["freddy-ai"] = 8_000` — config + integration + training Freddy AI sur 1 produit (Copilot OU Agent OU Insights).

### Volume contenu (~30 cles i18n × 2 langues)

Sections cibles :
- Hero (h1 "Freddy AI : votre copilot, agent et analyste IA" + subtitle + CTA)
- "3 produits, 3 usages" :
  - Copilot (assistance agents L1/L2) — citation Salesforce
  - Agent (deflection autonome L1) — citation Freshservice Benchmark 34%
  - Insights (analytics conversationnel) — citation BCG +18 pts CSAT
- "Resultats prouves" (block trust avec sources)
- "Cas d'usage par segment" (ITSM / CX / ESM, 1 cas chacun)
- "Pour qui" (DSI, Lead Transfo, Service Manager)
- FAQ (4 Q/R sur AI : RGPD, hallucination, ROI, training)
- CTA final

### Sources analystes obligatoires

3 sources nommees :
1. **BCG Agentic AI 2025** : +50% productivite agents, +18 pts CSAT
2. **Salesforce State of Service 2025** : ROI moyen IA support (chiffres a confirmer via agent Explore)
3. **Freshservice Benchmark 2025** : 34% deflection avec Freddy Agent

Citation visible : `<small>Source : BCG Agentic AI 2025</small>` ou `cite` attribute. Pas de chiffre sans source.

### Agent Explore prealable

Au demarrage de la story : invoker agent Explore sur `/Users/WhatAService/Downloads/Reports/` pour extraire les chiffres exacts. Pattern S20 K5.

Volume contenu legerement superieur a US-S21-1 (~30 vs ~25 cles) MAIS pattern code path identique. Sensibilite des claims AI est mitigee par l'usage strict des sources analystes deja synthese en S20.
