## Notes techniques / decisions

### Methode de verification triple

Avant suppression, executer pour CHAQUE export :

```bash
# 1. Recherche dans src/
grep -rn "DEMOGRAPHIC_FIELDS" src/ --include="*.ts" --include="*.tsx"

# 2. Recherche dans tests/
grep -rn "DEMOGRAPHIC_FIELDS" tests/ --include="*.ts" --include="*.tsx"

# 3. Recherche dans node_modules/.cache si Next.js prerend
# (TypeScript strict + npm run build = filet final)
```

Si une seule occurrence non-declarative apparait → la story est mise en pause et le PO arbitre.

### Risque de regression

**Faible** — TypeScript en mode strict (`tsconfig.json`) detecte tout import casse au build. Le filet de securite est dur.

### Pourquoi 1 pt

Pure suppression, scope chirurgical, methode mecanique. Quick win parfait pour le buffer S19.
