## User Story

```
En tant que DSI ou Lead Transfo recherchant "audit Freshservice" ou "optimisation Freshdesk" sur Google,
je veux atterrir sur une page dediee qui presente l'offre Health Check WaS,
afin de comprendre le livrable (rapport + roadmap) et reserver un audit en 24h.
```

## Notes techniques / decisions

### Prix reference (12k/8k/6k €)

`SERVICE_PRICE_FROM["audit-optimisation"] = 6_000` — Health Check 2 semaines, livrable rapport + roadmap actionnable.

### Volume contenu

~20 cles i18n × 2 langues = 40 entrees. Sections cibles :
- Hero (h1 + subtitle + CTA)
- "Ce que vous obtenez" (3 livrables : rapport, roadmap, presentation)
- "Methodologie 2 semaines" (semaine 1 audit + semaine 2 recommandations)
- "Pour qui" (DSI, Lead Transfo, RUN manager)
- FAQ (3 Q/R)
- CTA final

### Sources analystes a citer

- Freshservice Benchmark 2025 (criteres comparatifs avant/apres optimisation)

### Pourquoi 1 pt et pas 1.5

Volume contenu plus simple que Migration ServiceNow (US-S21-1, 2 pts) car pas de section "garanties zero perte donnees" ni "checklist 10 etapes". Le pattern code path est deja livre (US-S20-2).
