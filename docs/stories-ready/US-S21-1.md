
## Notes techniques / decisions

### Prix reference

`SERVICE_PRICE_FROM["migration"] = 12_000` — mission lourde (due diligence + migration + change management).

### Volume contenu (~25 cles i18n × 2 langues)

Sections cibles :
- Hero (h1 "Migration ServiceNow → Freshworks" + subtitle + CTA)
- "Vos doutes" (3 cas d'echec ServiceNow concrets : perte historique, downtime, regression workflows)
- "Notre methode 4 etapes" (audit + mapping + migration incrementale + formation)
- "Garanties" (zero perte donnees, zero downtime planifie, retour arriere documente)
- "Pour qui" (DSI ETI 200-2000 collab avec ServiceNow legacy)
- Trust block (Forrester Wave Strong Performer + ITIL 4 + cas WaS si dispo)
- FAQ (4 Q/R)
- CTA final

Forrester Wave ESM Q4 2025 (deja exploitee S20). Citation dans un block trust visible. Si possible, ajouter une 2eme source comme Forrester TEI Freshworks 2024 (deja chargee pour US-S20-7).

### Pourquoi 2 pts

Volume contenu (~25 cles vs ~20 pour US-S21-3) + section sensible "garanties zero perte" (verbatim a verifier juridiquement) + sources analystes a citer precisement.
